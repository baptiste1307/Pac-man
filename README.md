## Instruction

### Requirements
- Python 3.10 and above
- uv dependency manager

### Installation ⚙️
To create a virtual environment for the project, and have necessary packages installed, use:
> make install

or directly:
> uv sync

### Launch 🚀
To execute the program and play with the mini-game, use:
> make run

You can also directly type the execution command in Terminal:
> uv run python3 -m src

### Debug 🕵🏻‍♂️
The command below allows you to use python built-in debugger pdb:
> make debug

### Lint ✨
Apply different modes for linting with flake8 and mypy:
> make lint

> make lint-strict

### Cleanup 🚮
> make clean
